A brief concept of programming languages

A programming language is a notation designed to connect instructions to a computer or machine. Programming languages are primarily used to monitor a machine 's output, or to express algorithms. There are already thousands of programming languages to be introduced. Many languages must be specified in an imperative form in the computer field whereas other programming languages use declarative form. The software can be divided into two types such as syntax and semanthetic. Some languages are characterized by a standard of SO such as the C language.
